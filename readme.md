## SupMan

SupMan is a Support Manager dashboard for Sonar made with love, and [Laravel](https://laravel.com/docs/10.x/).

The [quickphp](https://codeberg.org/gomarcd/quickphp) docker stack is used as a starting point. [Guzzle](https://docs.guzzlephp.org/en/stable/overview.html) is used to make API calls, and [Filament](https://filamentphp.com/docs/2.x/admin/installation) to visualize the data in widgets made with [Livewire](https://laravel-livewire.com/).
